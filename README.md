# 儿童晨检机器人--个人端

## 整体
使用Vue框架，以及mint-ui与MUI组件，数据接口采用 easymock伪接口
页面风格:整体采用黄色#ffc000为背景色

## home界面
###轮播图
使用mint-UI的swiper的组件，使用时调用即可
###中间导航栏
6项导航栏 
主要功能分别为：
+ 体检报告，通过路由进入后完成一个体检表的展示
+ 数据统计，学生体检信息，近两周的展示，采用折线图
+ 可疑情况， 可疑情况展示
+ 接送情况
+ 天气帮手，可以查看天气预报
+ 产品介绍，介绍儿童智能晨检机器人的相关情况
4项底部导航栏
主要功能分别为：
+ 首页
+ 资讯分享
+ 家校互动
+ 个人中心
